CSS 스타일 선택자 우선순위

    기본적으로 css 속성은 style 태그 또는 css 파일에 작성된 순서(위에서 아래로)로
    해석이 진행되지만 같은 요소에 여러 css 속성이 설정되는 경우 우선순위에 따라 적용된다

    1. 동일한 우선순위의 css 속성 -> 제일 마지막에 작성된 css 속성 반영
    2. 여러 선택자를 이용해 같은 요소를 선택하더라도 지정되는 css 속성이 다르면 모두 반영
    3. 태그와 style이 가까이 위치할수록 우선순위 적용
    4. css 파일과 style 태그에 작성된 속성은 동일한 우선순위를 가지나 가장 최근에 작성한 스타일 적용
      ex) link rel="stylesheet" href="스타일.css"
          style
            /* css 설정 값 작성 */
          /style
          마지막에 작성한 "style 태그" 적용

      ex) style
            /* css 설정 값 작성 */
          /style
          link rel="stylesheet" href="스타일.css"
          마지막에 작성한 "link 태그" 적용

    1순위 : css 속성 : 속성값 !important;
    2순위 : inline-style 속성 -> p style="font-size: 20px" 처럼 태그에 속성으로 주는 것
    3순위 : #아이디 속성명 #h1{ css 설정 값 }
    4순위 : .클래스 속성명 -> 주로 반복적인 디자인 적용, 대표적인 방법 -> .h1{css 설정 값 }
    5순위 : 태그이름 -> h1{ css 설정 값 }
