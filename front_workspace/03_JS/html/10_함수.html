함수(function)

      - 목표하는 결과를 만들기 위한 코드의 묶음
      - 코드를 {} (중괄호, 블록)으로 묶어 작성하고 붙여진 이름을 호출해서 사용

      [기본 함수 형태]
      function 함수명() { <- 함수 이름 선언
        함수 정의
      }
      -------------------------------
      함수명() 함수 호출

      함수 : 특정 행동을 동작하고자 할 때 그 행동을 하기 위한 여러 기능들을 묶어 하나의 알고리즘을 만드는 것

      예) function 전자레인지작동("몇 초") {
          1. 전자레인지 코드를 꼽는다.
          2. 전자레인지에 음식을 넣는다.
          3. 전자레인지에서 ("몇 초") 동안 돌리기 위해 시간 버튼을 클릭한다.
          4. 전자레인지 시작 버튼을 누른다.
          5. 전자레인지가 ("몇 초") 동안 돌아간 후 종료한다.
      }

매개 변수(parameter)

      중간 역할을 하거나 전달하는 변수
      - 함수 선언 시 () 안에 작성되는 변수
      -> 전달 인자로 부터 받은 값을 저장
      -> 함수 정의 부분에서 사용 가능

      parameter : 변경이 될 기준을 제공

전달인자(argument)

      * 傳達 引子(전달 인자, Arguments) //자바에서 설명
      傳 전할     전 정보를 전하다
      達 도달될   달 전달의 완료 또는 목적지에 도착했다
      引 끌어당길 인 무언가를 특정 위치로 이끈다
      子 아들     자 특정한 목적이나 역할을 가진 요소
      
      Argument : 제공되는 정보, 실제 값
      함수(function)가 호출 될 때 아규먼트의 값들이 증거처럼 파라미터에 전달

매개변수와 전달인자

return (반환)

      - 현재 수행중인 함수를 종료하고 호출한 곳으로 돌아감
      
      - 호출한 곳으로 돌아갈 때
        return 옆에 작성된 값으로 돌아갈 수 있다.

        만약에 return 100; : 호출한 곳으로 100을 가지고 돌아감
