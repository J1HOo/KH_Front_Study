img 태그 사용하기

      - 웹 문서에서 사진이나 그림 같은 이미지를 삽입하는 용도의 태그

      [속성] 시작 태그 안에 작성하는 설정

        1. src : 삽입할 이미지의 경로를 작성하는 속성 (파일 위치, 인터넷 주소)
        2. width / heigh : 너비 높이를 조정하는 속성
        3. alt : 이미지에 대한 설명을 작성하는 속성
          - 이미지가 보이지 않을 경우 화면에 이미지 이름 출력
          - 스크린 리더가 이미지 설명을 읽어줌
            스크린 리더 : 화면의 텍스트를 읽어주는 기능

        이미지는 닫는 태그 없이 단독으로 한 번만 작성
