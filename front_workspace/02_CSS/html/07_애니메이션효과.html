transition 과 transform

transition 속성

      transition : 요소의 속성이 변경될 때 그 변화가 즉각적으로 나타나는 대신,
      부드럽게 전환되는 애니메이션 효과 추가
      
      transition : 속성명/ 전환시간/ 전환속도곡선/ 전환이 시작되기 전 지연시 대기했다가 
      
      ex) 3초 후에 배경색을 빨간색으로 변경, 변화 속도는 0.5초 속도로 일정한 속도를 유지하며 변환
      transition: background- color 0.5s linear 3s;
      속도와 몇 초 후 시작하라는 표기는 자주 사용되지 않음
      (hover, active 같은 효과에서 색을 지정하는 것이지 transition 내에서 색상 지정을 하는 것은 아님)

transform 속성

      transform : 요소를 이동, 회전, 크기 조절, 뒤집기 등의 변화를 적용하는 속성
      1. translate : 요소를 X, Y 축 방향으로 이동
      transform: translate(10px, 20px); -> 가로 오른쪽 10px, 세로 아래쪽 20px 이동
      transform: rotate(45deg); -> 45도 회전
      transform: scale(1.5) -> 요소의 크기를 1.5배 키움
      transform: skew(10deg, 5deg) -> 가로 오른쪽 10도, 세로 오른쪽 5도 기울이기
      
      ex) 요소를 오른쪽으로 20px, 아래로 10px, 크기를 1.2배 키우기
      transform 기능을 2가지 이상 사용할 때 작성하는 방법
      
      transform:translate(20px, 10px) scale(1.5); 
      띄어쓰기로 구분하며 가지 수는 제한이 없다
