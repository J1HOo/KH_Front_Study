AND 연산자 (A && B)

        - A, B 모두 true 일 경우 결과로 true를 반환하는 연산자
          (나머지 경우는 모두 false)
          A = 조건 1     B = 조건 2
          A   와   B가 모두 참일 때만 true
        (내가 회원가입 할 때 작성한 값이 로그인을 시도했을 때)
          A = 아이디가 일치하는가 ?
          B = 비밀번호가 일치하는가 ?

로그인 하기

        - 회원가입 할 때 저장한 아 이 디 : 'member01'
        - 회원가입 할 때 저장한 비밀번호 : 'pass01'

OR 연산자(||)

      - A, B 둘 중 하나라도 true 일 경우 결과로 true 반환하는 연산자
        (A, B 모두 false 일 경우에만 false)

        회원가입 할 때 전화번호 나 휴대폰 번호 둘중 하나라도 입력
        A = 핸드폰 번호를 작성했는가
        B = 일반 전화번호를 작성했는가

논리 부정(NOT) 연산자(!)

      - 논리 값을 반대로 바꾸는 연산자
      - 값이 거짓일 때 true를 반환

      ex) const 사과 = "apple";
          사과 !== "apple"? "true" : "false"
          결과 : false

부등호 (=,==, ===)

      = (할당 연산자)
        - 변수에 값(숫자나 문자 등)을 할당할 때 사용
        ex) const a = 5; let 사과 = "빨간색";

      == (동등 연산자)
        - 값만 비교하는 연산자, 타입이 다르더라도 값이 같으면 true를 반환
        const 글자5 = "5" == const 숫자5 = 5 ? "true" : "false";
        결과 : true

      === (일치 연산자)
        - 값과 타입 모두 비교하는 연산자, 값이 같아도 타입이 다르면 false를 반환
        const 글자5 = "5" === const 숫자5 = 5 ? "true" : "false";
        결과 : false
