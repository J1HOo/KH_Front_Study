첫글자가 대문자로 시작하는 자바스크립트

      Math, Array, Date 와 같이 영어로 시작하는 글자의 첫 글자가 대문자로 시작하는 단어는
      자바스크립트에서 파일을 만들어서 기능을 제공하거나 개발자가 외부에서 사용할 수 있는 파일을
      만들어서 외부에서 사용 가능하도록 표기하는 것

      주로 React, Java에서 많이 사용

      개발자의 눈에는 보이지 않지만 Math.js, Array.js, Date.js 와 같은
      파일로 만들어진 기능을 가져다 쓰겠다는 표현

      파일명 자체에서 첫글자를 대문자로 작성 -> 외부에서 사용하겠다라는 의미가 포함

      파일명 자체에서 첫글자를 소문자로 작성해준다면 외부에서 사용 없이 내부에서 사용한다는 의미
