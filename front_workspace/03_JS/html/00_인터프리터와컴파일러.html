인터프리터(interpreter)와 컴파일러(compiler)

    * 프로그래밍 언어로 작성한 코드를 기계어로 번역 해주는 소프트웨어

    컴파일러(compiler) : 프로그래밍 언어로 코딩된 소스 코드를 컴퓨터가 알아들을 수 있는 언어로 변환하여 파일을 생성 -> 그 파일 실행
                      번역에 시간이 소요되지만 실행이 빠르다는 장점이 있다
                      예시) C, C++, Java
    인터프리터(interpreter) : 소스 코드를 한 줄씩 해석하면서 실행
                          번역 과정을 생략할 수 있지만 한 줄씩 해석하면서 실행하기 때문에 컴파일러 방식에 비해 속도가 느림
                          예시) Python, JavaScript, Ruby
