label 태그

      속성 선택자 css 문법
      for 속성의 값이 input 요소와 연결하는 역할
      input에서 id 값과 for와 속성값이 같으면 연결 또는  label안에 input 태그를 작성할 수 있음
      label안에 input 태그 작성시 for 생략 가능

      파일 선택 버튼을 누르지 않고 사진 선택하기를 눌러 파일 선택하는 창과 연동해서 사용 가능

      주로 input type="file" 속성은 히든 처리해서 화면에서 숨긴다.
