테이블 관련 추가 태그

    [테이블 영역 지정 태그]
    thead 태그 : 테이블 상단 영역(컬럼의 제목을 작성하는 공간)
    tbody 태그 : 테이블 중단 영역(컬럼 안에 내용이나 값을 작성하는 공간)
    tfoot 태그 : 테이블 하단 영역(컬럼 특정 값의 합계나 인원수를 작성하는 공간)

    (* 참고 사항 *)
    table 태그 내부를 thead, tboody, tfoot으로 나누지 않으면
    모든 tr, td, th 태그는 tbody 내부에 작성된 것으로 취급한다
