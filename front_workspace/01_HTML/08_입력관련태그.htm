form 태그 / input 태그

form 태그

      - 입력 양식을 작성하는 공간
      - 내부에 입력된 값을 지정된 곳으로 제출하는 역할

      [속성]
      1)action : 제출될 곳을 지정하는 속성(제출할 주소 설정) 
      2)method(방법) : 설정된 곳으로 데이터를 전달하는 방식
      3)name : 이름 지정하는 속성(js와 함께 사용)
      4)target : 제출된 페이지를 현재/새 탭으로 열기 설정

input 태그

      - 웹 문서에서 사용자(클라이언트)가 입력할 수 있는 양식을 제공하는 태그
      - 많은 type(속성)이 존재하고, input의 변형 형태도 다수 존재

      *input 태그에는 name 속성이 작성되어야할 경우가 상당히 많음
      name 태그를 사용해 이름끼리 연결해서 사용

textarea 태그

      여러 줄의 텍스트를 입력할 수 있는 태그
      
      [속성]
      rows : 텍스트 영역에서 기본 줄 수 지정
      rows : 텍스트 영역에서 기본 가로 너비 수 지정
      placeholder : 사용자가 입력하기 전에 나타나는 힌트 텍스트
      maxlength : 입력 가능한 최대 문자수 설정
      readonly : 읽기 전용으로 설정되어 입력 불가능한 칸을 만듦
      disabled : 비활성화된 상태로 만듦
