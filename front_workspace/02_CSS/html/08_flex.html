Flex 레이아웃

      display : flex:
      div안에 작성한 태그들을 유연하게 배치 할 수 있는 속성
      제일 많이 쓰이는 기능
      수평이나 수직 정렬 가능
      flex의 경우 div와 같은 block(한 줄 차지) 형태를 inline(필요한 만큼만 차지) 형태로 바꿔준다

      같이 자주 쓰이는 속성
      1. 가로로 존재하는 태그의 여백을 설정하는 방법

      1-1. justify-content: center;
        flex 컨테이너 내의 자식 태그를 가로방향으로 가운데 정렬

      1-2. justify-content:space-around;
        flex 컨테이너 내에서 자식 태그들 사이에 균등한 여백을 주어 배치

      2. 세로로 존재하는 태그 줄의 여백을 가운데로 설정하는 방법

      2-1. align-items: center;
        세로 줄에서 가운데 정렬
        주로 flex-direction:row; 와 함께 사용 할 때 세로 정렬이 필요한 경우 사용
      
      3. 태그의 너비가 한 줄을 넘어가면 자동 줄바꿈이 되는 태그

      3-1. flex-wrap : wrap;
        가로에서 세로로 줄바꿈 되어 넘어가지 않도록 설정
        너비가 넘어갈 경우 자동 줄바꿈

flex와 inline 차이

특징 요소 배치 방식 크기 제어 정렬 옵션 응용 범위
inline 같은 공간에 나열되며 필요한 공간만 차지 높이, 너비 지정 어려움 정렬 옵션 제한적 text, link 요소 등에 사용
flex 가로,세로 방향으로 유연하게 배치 크기, 여백 쉽게 제어 가능 다양한 정렬 옵션 지원 전체 레이아웃 설계 주로 사용
