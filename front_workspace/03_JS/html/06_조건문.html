조건문(Condition)

    Condition = 어떤 조건,상태에 따라 다르게 동작하는 것을 의미
                조건 = 만약 이 조건이 참이라면 이 작업을 실행
                상태 상황 = 특정한 상태나 상황에서만 이 작업 실행
      프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류로
      작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

if(만약..)

      
        if(조건식) {
          조건식이 true인 경우 실행할 코드;
        }
      
      * 조건식 : 결과가 true, false 인 식(연산)
      * 조건문 : 조건식의 결과에 따라 실행을 제어하는 구문(if, switch)

입력받은 수가 양수인지 아닌지 검사

if(만약에) - else(아니면)

      
        if(조건식){
          조건식이 true인 경우 실행할 코드;
        } else {
          if에 작성한 조건식이 false 인 경우 수행할 코드
        }
        if-else : 조건에 따른 행동 케이스가 많아야 하는 경우
        삼항연산자 : 케이스를 둘 중 하나로 택1 하는 정도의 경우

        else의 (조건식)은 존재 X : if 조건식을 따라감

if(조건식) - else(아니면) - else if(이것도 저것도 아니라면)

      
        if(조건식A) {
          조건식A가 true 인 경우 실행;
          true
        } else if(조건식B) {
          조건식A가 false 이면서
          조건식B가 true 인 경우 실행;
        } else {
          조건식A 조건식B 모두 false 인 경우 실행;
        }

switch 문

switch, case, default 문

      switch 문
      - 식 하나의 결과로 많은 경우의 수를 처리 할 때 사용하는 조건문  
      
        switch(식) { // 식의 결과는 정수 또는 문자열
          case 결과1:
          결과 1일 때 수행할 코드;
          break; 결과 1이 맞을 때 멈추고 싶다.
          
          case 결과2:
          결과 2일 때 수행할 코드;
          break; 결과 2이 맞을 때 멈추고 싶다.
          case 결과3:
          결과 3일 때 수행할 코드;
          break; 결과 1이 맞을 때 멈추고 싶다.
          
          ...
          
          default:
          위 모든 케이스에 만족하지 못할 때 수행할 코드
          (if계의 else와 같음)
        }

조건이 여러개인 switch case 문

      if문처럼 조건이 여러 경우 일 때는
      case 조건1: case 조건2: case 조건3:
        조건1이나 조건2나 조건3일 때 부합할 경우 실행할 코드
      case 조건4: case 조건5:
        조건4나 조건5일 때 부합할 경우 실행할 코드

      default:
      조건1~조건5까지 모두 부합하지 않을 경우 실행
